# Partner Targets

## Integration Roadmap

### 1. **Dropbox**
- **Rationale**: Dropbox is a widely used cloud storage service. Integrating with Dropbox will allow users to seamlessly transfer files from their Dropbox accounts to File Share Guard.
- **Free-Tier Limits**: 2GB storage, 50GB monthly bandwidth.
- **Integration Effort**: Medium (M)
- **Value-Add**: Solves the user job of secure file sharing from existing cloud storage solutions.
- **Affiliate/Revenue-Share**: Yes, Dropbox offers an affiliate program.
