# Customer Journey Map
## Aware
### Trigger Event
- User receives a file from a colleague or client via email
- User searches for a reliable file transfer solution online

### Friction Points
- Difficulty finding a lightweight, self-hosted solution
- Concerns about security and data protection

### User Emotions
- Frustration
- Anxiety

### Opportunities to Delight
- Provide a simple, one-click installation process
- Offer a free trial or demo to alleviate security concerns

### Metric
- Time to find a suitable file transfer solution (TTFS)
- Number of potential customers who abandon their search (PCA)

## Consider
### Trigger Event
- User discovers file-share-guard through online search or social media
- User compares file-share-guard with other file transfer solutions

### Friction Points
- Difficulty understanding the features and benefits of file-share-guard
- Concerns about the security and reliability of file-share-guard

### User Emotions
- Curiosity
- Skepticism

### Opportunities to Delight
- Create a clear and concise product description and documentation
- Offer a free trial or demo to showcase the features and benefits of file-share-guard

### Metric
- Number of users who engage with the product documentation (NPD)
- Number of users who request a demo or trial (NDT)

## Try
### Trigger Event
- User installs and configures file-share-guard
- User shares files using file-share-guard

### Friction Points
- Difficulty with installation and configuration
- Concerns about the security and reliability of file-share-guard

### User Emotions
- Frustration
- Relief

### Opportunities to Delight
- Provide a simple and intuitive installation process
- Offer timely and effective support to alleviate concerns

### Metric
- Time to install and configure file-share-guard (TIC)
- Number of users who report issues or concerns (NRI)

## Adopt
### Trigger Event
- User successfully shares files using file-share-guard
- User recommends file-share-guard to colleagues or clients

### Friction Points
- Difficulty with scalability and performance
- Concerns about the long-term viability of file-share-guard

### User Emotions
- Satisfaction
- Loyalty

### Opportunities to Delight
- Offer scalable and performant solutions to meet growing needs
- Provide transparent and regular updates to alleviate concerns about long-term viability

### Metric
- Number of users who recommend file-share-guard (NRM)
- Net Promoter Score (NPS)

## Expand
### Trigger Event
- User expands their use of file-share-guard to new teams or departments
- User integrates file-share-guard with other tools and services

### Friction Points
- Difficulty with integration and customization
- Concerns about the cost and complexity of expansion

### User Emotions
- Excitement
- Confidence

### Opportunities to Delight
- Offer seamless integration with other tools and services
- Provide transparent and flexible pricing to alleviate concerns about cost and complexity

### Metric
- Number of users who expand their use of file-share-guard (NEU)
- Revenue growth rate (RGR)
