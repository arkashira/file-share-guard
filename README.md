# File Share Guard
Secure file sharing with end-to-end encryption.

## Usage
1. Upload a file: `python src/file_share_guard.py upload example.txt`
2. Download a file: `python src/file_share_guard.py download example.txt`

## Requirements
* Python 3.8+
* No external dependencies required

## Tests
Run tests using `pytest`: `pytest tests/test_file_share_guard.py`
